# 成大新陳代謝科住院醫師評分系統

> 供成大新陳代謝科主治醫師使用的住院醫師定期評估追蹤工具。

---

## 快速開始

### 本機開啟
直接用瀏覽器開啟 `index.html` 即可使用（不需要伺服器）。

### 部署到 GitHub Pages
1. 建立一個新的 GitHub Repository
2. 將 `index.html` 上傳至 repo 根目錄
3. 進入 repo → Settings → Pages
4. Source 選擇 `main` branch → `/ (root)` → Save
5. 稍等片刻，即可透過 `https://<你的帳號>.github.io/<repo名稱>` 存取

## 功能說明

| 頁面 | 功能 |
|------|------|
| **Dashboard** | 顯示現役學員評估提醒燈號（紅/黃/綠）、雷達圖、近6個月趨勢 |
| **學員管理** | 新增學員、查看現役與歷史學員列表、結束評估（封存） |
| **填寫評估** | 11項計分（1–10滑桿）+ 4項勾選 + 文字評語 |
| **歷史紀錄** | 選擇學員查看所有評估紀錄、折線趨勢圖、橫向柱狀圖 |
| **統計分析** | 各屆均分、項目雷達、月評估量、達成率、分數分佈 |

### 評估項目

**計分項目（1–10）**
- 臨床知識：糖尿病診斷與藥物調整、甲狀腺疾病評估與處置、骨質疏鬆症診斷與治療、代謝症候群與肥胖管理、檢驗數值判讀
- 臨床技能：病史詢問完整性、鑑別診斷能力、治療計畫制定與說明
- 態度溝通：病患溝通與衛教、病歷書寫品質、團隊合作與交班

**勾選項目（達成 / 未達成）**
- 完成糖尿病個案報告
- 參與門診 ≥ 20 次
- 完成甲狀腺超音波判讀訓練
- 參與科內教學討論

### 提醒機制
- 🔴 紅燈：距上次評估 ≥ 30 天（需立即評估）
- 🟡 黃燈：距上次評估 ≥ 20 天（即將到期）
- 🟢 綠燈：距上次評估 < 20 天（正常）

### 資料儲存
- 目前版本使用瀏覽器 **localStorage** 儲存資料
- 資料存於同一瀏覽器，換裝置或清除瀏覽器資料會遺失
- 未來版本計畫串接 Google Sheets
